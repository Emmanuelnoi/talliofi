# Talliofi User Manual

## 1. First-Time Setup

1. Open Talliofi and complete onboarding.
2. Enter gross income and tax assumptions.
3. Create budget buckets and target allocations.
4. Add recurring expenses.
5. Finish setup to create your first plan.

## 2. Main Sections

- Dashboard: monthly overview, key numbers, alerts, net worth summary.
- Income: update income amount/frequency.
- Taxes: simple effective rate or itemized components.
- Buckets: manage category allocations and rollover settings.
- Expenses: add/edit transactions and recurring templates.
- Goals: track savings and debt-payoff targets.
- Net Worth: maintain assets and liabilities.
- History: review monthly snapshots and trends.
- Reports: analyze spending and export CSV/PDF.
- Settings: preferences, import/export, privacy, security, demo tools.

## 3. Recommended Monthly Flow

1. Record or import recent expenses.
2. Check Dashboard surplus/deficit and alerts.
3. Review Reports for overspending categories.
4. Adjust bucket targets if spending patterns changed.
5. Update goals and net worth balances.

## 4. Demo Mode

- `/demo`: load standard demonstration data.
- `/demo?preset=heavy`: heavier spending/activity profile.
- `/demo?preset=investor`: investment-focused profile.

Use Settings -> Demo Tools to load/reset/remove demo data.

## 5. Data Safety

- Talliofi is local-first; your data stays on your device by default.
- Use export backups regularly.
- Use optional local encryption if needed.
- Use Danger Zone carefully; deletion is irreversible.
